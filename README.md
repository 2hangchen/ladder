# 梯子约拍小程序
## HELLOLADDER
梯子约拍是一个连接摄影师、摄影爱好者、模特以及普通人的摄影 o2o 约拍平台，主要提供全方位、个性化的约拍服务，用户可自主发起约拍或创作需求，通过系统推荐或自主选择符合要求的约拍的对象，与此同时为用户提供自我展示、学习交流、互动娱乐的平台，同时梯子约拍拥有强大的线下摄影师团队
。
